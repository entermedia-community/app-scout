Search Syntax

Here are common search terms and technique:

Multiple Words

When looking for multiple words the default is to AND together each word. Searching for big dog is the same as big AND dog.

You can combine multiple words together with the OR statement. For example: dog OR cat

Exact search

For example "hound dog" would find the only hound dogs.

Wild card searches

Multiple character wildcard searches looks for 0 or more characters. For example:

test* or te*t

The single character wildcard search looks for terms that match that with the single character replaced. For example:

te?t

Note: You cannot use a * or ? symbol as the first character of a search.

NOT

For example dog NOT hound would find any dog as long as it was not a hound dog.

Boosting

For unsorted searches use the caret, "^", symbol with a boost factor (a number) at the end of the term you are searching.
